</p></div><hr /><h3>📌 О проекте</h3><p>Этот проект создан для того, чтобы перевести абстрактные цифры военных бюджетов на язык человеческих возможностей. Мы показываем, сколько школ, больниц, университетов и научных центров можно было бы построить на средства, затрачиваемые на мировые конфликты.</p><p><b>Наша цель</b> — наглядная демонстрация альтернативных путей развития человечества.</p><h3>🌍 Поддержка языков</h3><p>Проект является международным и уже локализован на следующие языки:</p><ul><li>🇷🇺 <b>Русский</b> (<code>/ru</code>)</li><li>🇺🇦 <b>Украинский</b> (<code>/ukr</code>)</li><li>🇺🇸 <b>English</b> (<code>/en</code>)</li><li>🇪🇸 <b>Español</b> (<code>/es</code>)</li><li>🇩🇪 <b>Deutsch</b> (<code>/de</code>)</li><li>🇫🇷 <b>Français</b> (<code>/fr</code>)</li></ul><p><i>...и многие другие.</i></p><h3>🛠 Техническая реализация</h3><p>Проект построен на чистом <b>HTML/JS/CSS</b> для максимальной доступности и легкости встраивания (embed). Это позволяет запускать его на любом устройстве без сложной настройки.</p><ul><li>Каждая папка содержит локализованную версию калькулятора.</li><li>Логика расчетов основана на открытых данных о военных расходах и средней стоимости социальных объектов.</li></ul><h3>🤝 Как помочь проекту?</h3><p>Мы приветствуем любую помощь!</p><ul><li><b>Локализация:</b> Если вы нашли ошибку в переводе или хотите добавить новый язык — пожалуйста, ознакомьтесь с <a href="CONTRIBUTING.md">CONTRIBUTING.md</a>.</li><li><b>Данные:</b> Мы ищем актуальные статистические данные для обновления калькулятора (2024-2026 гг).</li><li><b>Обсуждения:</b> Присоединяйтесь к нашим <a href="https://www.google.com/search?q=https://github.com/peaceful-world-org/true-cost-of-war/discussions">Discussions</a>, чтобы предложить новые идеи.</li></ul><hr /><div align="center"><p><b>Peaceful World MTÜ</b> | Registry code: 80657458 | Tallinn, Estonia</p></div>
