📌 О проекте

Этот проект создан для того, чтобы перевести абстрактные цифры военных бюджетов на язык человеческих возможностей. Мы показываем, сколько школ, больниц и научных центров можно было бы построить на средства, затрачиваемые на мировые конфликты.

🌍 Поддержка языков

Проект является международным и уже локализован на следующие языки:

🛠 Техническая реализация

Проект построен на чистом HTML/JS/CSS для максимальной доступности и легкости встраивания (embed).

Каждая папка содержит локализованную версию калькулятора.

Логика расчетов основана на открытых данных о военных расходах и средней стоимости социальных объектов.
